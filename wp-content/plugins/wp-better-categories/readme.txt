== Description ==

Simple Visitor Counter is a simple plugin that embeds a simple visitor counter on your blog showing how many visitors your site has.

Our visitor counter plugin allows you to choose from a number of different styles, it can be embedded in the footer or activated as a sidebar widget.
By default it will display in your footer, to change these settings simply activate it as a widget in your themes appearance section and change the options there.

== Installation ==


This section describes how to install the plugin and get it working.

e.g.

1. Upload all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the appearance section and select widgets, select wordpress hit counter and configure from here.
